Condiciones para aplicar

Por los momentos no tenemos oficinas disponibles, por lo que los candidatos a aplicar trabajaran de remota. Una vez se forme el equipo de programadores, se procedera a buscar un local, preferiblemente en la ciudad de valencia. Bajo estas condiciones se deberan cumplir ciertas terminos:

Test

Para aplicar para la posicion de programador deberan realizar un test relativamente sencillo el cual medida sus habilidades en programacion

Horario

Por estar trabajando de manera remota, no tendran ningun horario en especifico, pero tendran que cumplir metas durante cada semana. Deberan estar disponibles entre las 3:30pm - 630 pm.

Control De Versiones (GIT)

Para el desarrollo de software es imprecindible la utilizacion de sistemas de control de versiones. Entre las populare utilizares git. Tendran que empujar cambios al repositorio como minimo una vez cada dos dias.

Nuevas Herramientas

El candito debera estar dispuesto a aprender cualquier tipo de herramientas asi como nuevos lenguages de programacion.
