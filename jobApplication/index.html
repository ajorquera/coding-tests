JorTech

JorTech, es una empresa joven que tiene como misión desarrollar nuevas aplicaciones web de calidad. Para ello incorpora entre sus filas talento venezolano capacitado y dispuesto a emprender proyectos novedosos utilizando herramientas prácticas e innovadoras.

Condiciones para aplicar

Actualmente no contamos infraestructura acondicionada para llevar a cabo las actividades, por lo que los candidatos a aplicar trabajarán en principio de manera remota. Una vez constituido el equipo de programadores se procederá a buscar una locación, preferiblemente en la ciudad de valencia. Los interesados en aplicar deben cumplir y adecuarse a condiciones y términos que a continuación se describen :

Test

Para aplicar a la vacante de programador el interesado debe realizar un test relativamente sencillo el cual medirá sus habilidades, conocimientos y destrezas en programación, siendo este un requisito imprescindible para continuar en el proceso de selección.

Test: https://github.com/jortechve/tests

Importante: evitar copiar cualquier código de internet. El test es lo suficientemente sencillo para crear su propia implementación. El candidato que haga caso omiso, no será tomado en cuenta.

Condiciones de trabajo - Horario

Por estar trabajando de manera remota el programador seleccionado no cumplirá, en principio, con un horario específico. Sin embargo deberá semanalmente cumplir con determinados objetivos y metas, para ello se implementarán diversas herramientas de evaluación que permitirán medir su desempeño y desenvolvimiento a lo largo del tiempo. Considerar que debe estar disponible entre las 3:30 pm - 6:30 pm, para reuniones o consultas vía web. Tomar en cuenta que deberán trabajar con cambio de horario con Inglaterra.

Control De Versiones (GIT)

Para el desarrollo de software es imprecindible la utilizacion de sistemas de control de versiones. Entre las populare utilizares git. Tendran que empujar cambios al repositorio como minimo una vez cada dos dias.

Nuevas Herramientas

El aplicante debe estar dispuesto y abierto a aprender cualquier tipo de herramienta, así como nuevos lenguajes de programación. Esto le permitirá crecer profesionalmente y dentro de la organización.

Aplicar

Para aplicar debe llenar el siguiente formulario: Una vez creado su usuario serán redirigidos a hipchat. HipChat es un servicio para crear un sistema de mensajería, con lo que nos permitirá llevar un seguimiento práctico y mayor facilidad a cada candidato que se postule. Además estará en contacto directo con las personas que le evaluarán, por lo que podrá despejar cualquier duda o inquietud permitiendo que sea más rápida y efectiva la respuesta de ambas partes.
